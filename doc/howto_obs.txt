Pour utiliser ReaChannelMap (qui fait partie de la suite ReaPlugs) dans OBS Studio afin de gérer vos canaux audio, voici la marche à suivre étape par étape.

1. Téléchargement et Installation
Allez sur www.reaper.fm/reaplugs/ et suivez ces instructions :

Lien : Descendez en bas de la page et cliquez sur "Download: ReaPlugs VST v2.36 64-bit" (la version 64-bit est indispensable pour les versions modernes d'OBS).

Installation : Lancez l'installateur .exe.

Important : Par défaut, il installe les fichiers dans C:\Program Files\VSTPlugins\ReaPlugs. C'est un emplacement standard qu'OBS scanne automatiquement.

Durant l'installation, assurez-vous que ReaJS est bien coché (ReaChannelMap est souvent inclus via l'interface ReaJS ou comme plugin autonome selon les versions
du bundle).

2. Configurer OBS Studio pour voir le plugin
Si OBS était ouvert pendant l'installation, redémarrez-le.

Dans le panneau Mélangeur audio d'OBS, cliquez sur les trois petits points (ou l'engrenage) à côté de la source audio que vous voulez modifier.

Sélectionnez Filtres.

Cliquez sur le + en bas à gauche et choisissez Plugin VST 2.x.

Donnez-lui un nom (ex: "Routing Canaux").

Dans le menu déroulant "Plugin VST 2.x", recherchez reajs ou reachannelmap.

Note : Si vous ne voyez que "reajs", sélectionnez-le, puis cliquez sur "Ouvrir l'interface du plugin" et cherchez le script "channel_mapper" ou "channelmap" à
l'intérieur.

3. Utiliser ReaChannelMap pour le routage
Une fois le plugin ouvert (cliquez sur Ouvrir l'interface du plugin dans OBS) :

Le principe : Vous verrez une grille. Les lignes représentent vos entrées (le son qui arrive) et les colonnes vos sorties (vers où le son va dans OBS).

Affectation : Cliquez sur les cases pour créer des liens. Par exemple, si vous voulez envoyer le canal 1 de votre interface vers les canaux 1 et 2 d'OBS (pour
transformer un signal mono en stéréo), cochez les cases correspondantes.

Utilité : C'est l'outil parfait si vous avez une interface audio avec beaucoup d'entrées et que vous voulez isoler uniquement l'entrée micro n°3 pour qu'elle
soit diffusée sur votre stream.

?? Astuce de dépannage
Si le plugin n'apparaît pas dans la liste d'OBS :

Vérifiez qu'il est bien installé dans C:\Program Files\Common Files\VST2 ou C:\Program Files\VSTPlugins.

Allez dans les Paramètres d'OBS > Audio et vérifiez que votre fréquence d'échantillonnage (44.1kHz ou 48kHz) correspond à celle de votre système Windows pour
éviter les conflits de driver.
